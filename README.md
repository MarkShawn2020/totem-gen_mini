## ✨ 特性

- 🎨 基于用户输入实时生成独特的个人图腾
- 📝 智能创作与图腾相匹配的诗偈
- 🎯 小程序原生体验
- 🔄 实时预览生成效果
- 📤 支持分享与保存

## 🛠 技术栈

- **框架**: [Taro](https://taro.zone/)
- **UI 组件**: [NutUI](https://nutui.jd.com/)
- **状态管理**: react-query
- **类型检查**: [TypeScript](https://www.typescriptlang.org/)
- **开发工具**:
  - [@cs-magic](https://github.com/cs-magic) 系列配置
  - pnpm
